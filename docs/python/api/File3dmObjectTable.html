File3dmObjectTable¶

class rhino3dm.File3dmObjectTable¶

Bases: object

Represents a simple object table for a file that is open externally. This class mimics Rhino.DocObjects.Tables.ObjectTable while providing external eccess to the file.

AddBrep(brep)¶

Adds a brep object to Rhino.

Args:
brep : A duplicate of this brep is added to Rhino.
Returns:
A unique identifier for the object.
AddCurve(curve)¶

Adds a curve object to the table.

Args:
curve : A curve to add.
Returns:
A unique identifier for the object.
AddLine(_from, to)¶

Adds a line object to Rhino.

Args:

Returns:
A unique identifier of new rhino object.
AddMesh(mesh)¶

Adds a mesh object to Rhino.

Args:

id of new object.
