class rhino3dm.DimAngular(Dimension)

Annotation representing AnnotationTypes.Angular and AnnotationTypes.Angular3pt.

Points

Gets list of Point3d that froms a polyline representing the dimension lines. The different points are given with the following key names:

  • 'centerpt' Point3d denoting the center of the angular dimension

  • 'defpt1' Point3d denoting the first definition point

  • 'defpt2' Point3d denoting the second definition point

  • 'arrowpt1' Point3d denoting the place of the tip of the first arrow head

  • 'arrowpt2' Point3d denoting the place of the tip of the second arrow head

  • 'dimlinept' Point3d denoting the dimension line start

  • 'textpt' Point3d denoting the text location of the annotation

Return type:

dict[str, Point3d]

GetDisplayLines(dimstyle)

Get a dictionary with the fields 'lines', 'arcs' and 'text_rect'.

The list contained by 'lines' is a list of Line that are all straight line work for this dimension.

The list contained by 'arcs' is a list of Arc that are all arc line work for this dimension.

The list contained by 'text_rect' constains a list of Point3d for the box containing the dimension text.

Parameters:

dimstyle (DimensionStyle) – DimensionStyle to use

Return type:

dict[str, list[Line|Arc|Point3d]]

class rhino3dm.DimRadial(Dimension)

Annotation representing AnnotationTypes.Radius and AnnotationTypes.Diameter.

Points

Gets list of Point3d that from the lienwork the dimension lines. The different points are given with the following key names:

  • 'centerpt' Point3d denoting the center of the angular dimension, here you would place the center mark

  • 'defpt1' Point3d denoting the first definition point

  • 'radiuspt' Point3d denoting the radius point

  • 'dimlinept' Point3d dimension line point

  • 'kneept' Point3d knee point in the dimension line

Return type:

dict[str, Point3d]

class rhino3dm.DimOrdinate(Dimension)

Annotation representing AnnotationTypes.Ordinate.

Points

Gets list of Point3d that from the linework of the dimension lines. The different points are given with the following key names:

  • 'basept' Point3d Base point from which the ordinate dimension is calculated

  • 'defpt' Point3d denoting the first definition point

  • 'leaderpt' Point3d denoting the leader point

  • 'kinkpt1' Point3d first kink point

  • 'kinkpt2' Point3d second kink point

Return type:

dict[str, Point3d]
